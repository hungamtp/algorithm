##BST : Binary Search Tree
FULL : left has 2 nodes or 0 node
Perfect : all leaves are at same level
Complete : all levels full except last level, last level filled from left to right


##Traversal
1. DFS
  - Inorder (LNR) : left, node, right
  - Preorder (NLR) : node, left, right
  - Postorder (LRN) : left, right, node
2. BFS : Level Order Traversal
